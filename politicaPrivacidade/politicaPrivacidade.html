Introdução

Obrigado por escolher fazer parte da nossa comunidade no Clock In It. Estamos comprometidos em proteger suas informações pessoais e seu direito à privacidade. Se você tiver quaisquer questões ou preocupações sobre esta política de privacidade ou nossas práticas com relação às suas informações pessoais, por favor, entre em contato conosco pelo e-mail policy@clockin.it.

Esta Política de Privacidade descreve como o Clockin.it ("nós", "nosso" ou "nossa") coleta, usa e disponibiliza informações em conexão com o uso do nosso aplicativo móvel Clockin.it Register (o "App"). Este App é especificamente projetado para uso por pessoal autorizado, como Gerentes de RH de nossas empresas clientes ("Clientes"), para registrar dados biométricos faciais de seus funcionários ("Funcionários") para uso dentro do sistema de controle de tempo Clockin.it.

Informações Que Coletamos

Coletamos informações necessárias para fornecer as funcionalidades de registro do App e gerenciar os dados biométricos dentro do ecossistema Clockin.it:

  • Informações Coletadas ou Fornecidas via o App:
    • Fotos dos Funcionários: A função principal do App envolve capturar ou fazer upload de fotos dos Funcionários. Isso requer permissão no app para acessar a Câmera ou Galeria de Fotos do dispositivo.
    • Dados Biométricos: Processamos as fotos dos Funcionários usando análise de reconhecimento biométrico para extrair os pontos faciais e gerar os modelos de dados biométricos correspondentes.
    • Informações da Conta do Gerente de RH: Informações básicas necessárias para fazer login e usar o App.
  • Informações Coletadas Automaticamente:
    • Informações do Dispositivo: Podemos coletar informações padrão do dispositivo do Gerente de RH, necessárias para funcionalidade e segurança do app.
    • Dados de Uso e Desempenho: Usamos ferramentas como Firebase Crashlytics para coletar relatórios de falhas e dados de uso agregados.
    • ID de Publicidade: O App solicita acesso ao ID de Publicidade do dispositivo para fins de análise.

Como Usamos Suas Informações

Usamos as informações coletadas para os seguintes propósitos específicos:

  • Registrar Dados Biométricos dos Funcionários
  • Habilitar Verificação Biométrica
  • Funcionalidade e Segurança do App
  • Melhoria do App
  • Suporte Técnico
  • Conformidade com Requisitos Legais

Acesso e Compartilhamento de Dados

O tratamento de informações, particularmente dados biométricos dos Funcionários, é estritamente controlado:

  • Armazenamento e Acesso: As fotos dos Funcionários e os modelos de dados biométricos são armazenados com segurança nos servidores centrais do Clockin.it.
  • Disponibilidade no Clockin.it: Os dados biométricos registrados são disponibilizados através de nossa plataforma segura para outros aplicativos Clockin.it autorizados.
  • Provedores de Serviço: Utilizamos provedores terceiros para funções como hospedagem em nuvem e análise de dados.
  • Requisitos Legais: Podemos divulgar informações se exigido por lei ou para proteger direitos e segurança.

Segurança dos Dados

Implementamos medidas técnicas e organizacionais apropriadas para proteger as informações biométricas sensíveis processadas através deste App contra acesso não autorizado, perda, uso indevido ou alteração.

Retenção de Dados

Os dados biométricos e fotos dos Funcionários são armazenados enquanto o Funcionário estiver ativamente empregado pelo Cliente. Após a notificação de demissão ou término de necessidade de acesso, estas informações serão permanentemente removidas de nossos sistemas dentro de 5 dias.

Seus Direitos e Escolhas

Os Gerentes de RH têm direitos sobre suas informações de conta e dados de uso. Os Funcionários têm direitos sobre suas informações pessoais, incluindo dados biométricos. Como o Cliente é o controlador dos dados, os Funcionários devem direcionar solicitações sobre seus dados principalmente ao departamento de RH de seu empregador.

Privacidade de Crianças

Este App não é destinado ao registro de indivíduos menores de 16 anos ou da idade mínima relevante para emprego. Os Clientes são responsáveis por garantir que tenham os consentimentos necessários ou base legal para registrar dados dos Funcionários.

Mudanças nesta Política

Podemos atualizar esta Política de Privacidade. Notificaremos os Clientes sobre mudanças significativas. O uso contínuo do App após as mudanças constitui aceitação da nova política.

Contato

Se você tiver dúvidas sobre esta política de privacidade, entre em contato conosco em policy@clockin.it. Funcionários devem entrar em contato primeiro com o departamento de RH de seu empregador para consultas sobre seus dados.
